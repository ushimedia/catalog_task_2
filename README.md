# README

## 開発言語
 * Ruby 3.0.1
 * Rails 6.0.4

## 実行手順
 $ git clone git@github.com:ushimedia/catalog_task_2.git  
 $ cd catalog_task_2  
 $ bundle install  
 $ rails db:create && rails db:migrate  
 $ rails s
